Lights

The lights lumps is a new level lump in Doom 64, and contains a list of RGB colors definition for lighting in map referred by sectors.

Structure

By default, the lookup starts at 256, because Doom 64 generates a table of 256 grayscale colors palette (from 0 to 255 values), followed by the actual RGB table.

Offset (bytes) Size (bytes) C type Description
0 1 uint8_t Red value
1 1 uint8_t Green value
2 1 uint8_t Blue value
3 1 uint8_t Padding (unused)
4 2 uint16_t Tag
