Subsectors

Subsectors (or SSECTORS in map structure) are the convex structures wich composes the sectors. They are generated on BSP generation of map, and are composed by a batch of segments.

Structure

The sector which the subsector refers is determined by looking at his first segment, this is done in level load for future operations.

Offset (bytes) Size (bytes) C type Description
0 2 int16_t Seg count
2 2 int16_t Index of first seg
