Vertexes

The vertexes are the basic structure element of all the Doom 64 maps, setting up all the linedefs conections and polygon shape of subsectors.

Structure

The vertexes position are stored as fixed_t (where from 32-bit value, the high 16-bit represents the integer part and the least 16-bit the decimal part) as a difference from original titles.

Offset (bytes) Size (bytes) C type Description
0 4 int32_t X position
4 4 int32_t Y position
